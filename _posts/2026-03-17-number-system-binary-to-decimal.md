## 1. 진법(Number System)이란 무엇인가

진법은 숫자를 표현하는 규칙이다.  
어떤 숫자 체계에서 몇 개의 숫자를 사용하고, 자리값이 어떤 기준으로 증가하는지를 정의한다.

대표적인 진법은 다음과 같다.

| 진법 | 사용 숫자 | 설명 |
| --- | --- | --- |
| 10진수 | 0 ~ 9 | 우리가 일상적으로 사용하는 숫자 체계 |
| 2진수 | 0, 1 | 컴퓨터가 사용하는 숫자 체계 |
| 8진수 | 0 ~ 7 | 과거 컴퓨터 시스템에서 사용 |
| 16진수 | 0 ~ 9, A ~ F | 메모리 주소, 색상 코드 등에서 사용 |

## 2. 진법의 핵심 원리

모든 진법은 같은 원리로 작동한다.

각 자리의 값은 `(진법의 밑수)^자리 위치`로 결정된다.

즉

```text
자리값 = base^position
```

예를 들어 10진수 숫자 `345`는 다음과 같이 해석할 수 있다.

```text
3 x 10^2
4 x 10^1
5 x 10^0
```

즉

```text
3 x 100
4 x 10
5 x 1
```

이다.

## 3. 이진수(Binary Number)

이진수는 `0`과 `1` 두 개의 숫자만 사용하는 진법이다.

즉

```text
base = 2
```

따라서 자리값은 다음과 같이 증가한다.

```text
2^0 -> 2^1 -> 2^2 -> 2^3 -> 2^4 -> ...
```

즉

```text
1 -> 2 -> 4 -> 8 -> 16 -> 32 -> 64 -> ...
```

## 4. 이진수 자리값 구조

이진수는 오른쪽 끝 자리부터 시작한다.

| 자리 위치 (오른쪽 기준) | 거듭제곱 | 자리값 |
| --- | --- | --- |
| 0번째 자리 | 2^0 | 1 |
| 1번째 자리 | 2^1 | 2 |
| 2번째 자리 | 2^2 | 4 |
| 3번째 자리 | 2^3 | 8 |
| 4번째 자리 | 2^4 | 16 |
| 5번째 자리 | 2^5 | 32 |
| 6번째 자리 | 2^6 | 64 |
| 7번째 자리 | 2^7 | 128 |

즉 왼쪽으로 갈 때마다 값이 2배씩 증가한다.

## 5. 이진수 -> 십진수 변환 방법

이진수를 십진수로 변환하는 방법은 매우 단순하다.

규칙은 다음과 같다.

1. 오른쪽부터 자리값을 매핑한다.
2. `1`이 있는 자리의 값만 더한다.

## 6. 변환 예시

예시 이진수는 `10110₂`이다.

자리값을 매핑하면 다음과 같다.

| 이진수 | 1 | 0 | 1 | 1 | 0 |
| --- | --- | --- | --- | --- | --- |
| 자리값 | 16 | 8 | 4 | 2 | 1 |

`1`이 있는 자리만 더하면 된다.

```text
16 + 4 + 2 = 22
```

따라서

```text
10110_2 = 22_10
```

## 7. 처음 이해했던 방식 vs 실제 원리

처음에는 이진수를 다음처럼 이해하기 쉽다.

```text
왼쪽부터
16 8 4 2 1
```

그리고

```text
1 + 8 + 16 + 32 = 57
```

같이 계산하려고 할 수 있다.

하지만 이 방식은 단순한 계산 표현 방식일 뿐이고, 실제 원리는 다음과 같다.

```text
자리값 = 2^n
n = 오른쪽에서 몇 번째 자리인가
```

즉 지수 기반 구조가 진짜 원리이다.

## 8. 진법의 본질

진법의 핵심은 자리값이 거듭제곱 구조로 증가한다는 것이다.

정리하면 다음과 같다.

| 진법 | 자리값 증가 방식 |
| --- | --- |
| 10진수 | 10^0, 10^1, 10^2 ... |
| 2진수 | 2^0, 2^1, 2^2 ... |
| 8진수 | 8^0, 8^1, 8^2 ... |
| 16진수 | 16^0, 16^1, 16^2 ... |

즉 진법의 차이는 아래 두 가지다.

- 사용하는 숫자의 개수(base)가 무엇인가
- 자리값이 base의 거듭제곱으로 증가한다

## 9. 핵심 정리

이진수를 십진수로 변환하는 규칙은 다음과 같다.

1. 오른쪽 자리부터 시작한다.
2. 오른쪽 자리는 항상 `2^0 = 1`이다.
3. 왼쪽으로 갈 때마다 2배씩 증가한다.
4. `1`이 있는 자리값만 더한다.

## 10. 한 줄 요약

이진수는 각 자리값이 2의 거듭제곱으로 구성된 숫자 체계이며, `1`이 있는 자리의 값만 더하면 십진수로 변환할 수 있다.
