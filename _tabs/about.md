안녕하세요. 이곳은 데이터를 읽고, 정리하고, 해석하는 과정을 차근차근 기록하는 개인 데이터 분석 블로그입니다.

주로 아래 내용을 다룹니다.

- SQL 쿼리 풀이와 데이터 추출 과정
- 데이터 전처리, 탐색적 분석, 시각화 실습
- 분석 문제 해결 과정과 시행착오 기록
- 실무와 학습에서 얻은 인사이트 정리

완성된 결과만 남기기보다, 왜 이런 방식으로 접근했는지와 어떤 기준으로 판단했는지를 함께 적으려고 합니다. 같은 주제를 다시 보더라도 흐름을 빠르게 되짚을 수 있는 기록을 만드는 것이 목표입니다.

데이터 분석을 공부하는 분들, 문제 해결 과정을 함께 보고 싶은 분들께 도움이 되는 공간이 되었으면 합니다.
